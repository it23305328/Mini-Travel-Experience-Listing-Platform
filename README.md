# Mini Travel Experience

A MERN stack application for a seamless travel experience.

## Project Structure

```text
/
├── backend/            # Express.js server, MongoDB models, and API logic
└── frontend/           # React.js application (Vite)
```

## Getting Started

### Backend Setup
1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure your environment variables in `.env`.
4. Start the server:
   ```bash
   npm run dev
   ```

### Frontend Setup
1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```

## Technologies Used
- **MongoDB**: Database
- **Express.js**: Backend framework
- **React.js**: Frontend library (via Vite)
- **Node.js**: Runtime environment
- **JWT**: Authentication
- **Mongoose**: MongoDB object modeling
